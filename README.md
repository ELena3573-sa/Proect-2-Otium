# Proect-2 Постановка задачи разработчику для стримингового сервиса Otium.
Otium - это учебный проект, разработанный в рамках курса "Системный аналитик".
В стриминговый сервис Otium решили добавить новую функцию: рейтинг от кинокритиков и возможность сортировки.

Как системный аналитик, я проанализировала интервью заказчика, в draw.io добавила на диаграмму последовательности элементы, которые помогут отобразить процесс фильтрации и последующей сортировки списка фильмов и сериалов. 
Изучила коллекцию существующих запросов REST API веб-сервера и сервисов «Фильмы» и «Сериалы». Определила, какие API нужно доработать согласно диаграмме последовательности. Заполнила тикет на разработку.
